Easy fMRI employs advanced machine learning techniques and high-performance computing for analyzing task-based fMRI datasets. It provides a friendly GUI-based environment for applying Feature Analysis, Hyperalignment, Multi-voxel Pattern Analysis (MVPA), Representational Similarity Analysis (RSA), and more. Further, easy fMRI is integrated with FSL (for preprocessing steps), SciKit-Learn (for model analysis), PyTorch (for deep approaches), and AFNI and SUMA (for 3D visualization).
